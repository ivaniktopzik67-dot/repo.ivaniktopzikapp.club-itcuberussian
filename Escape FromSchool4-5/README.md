# Побег из школы 4–5: Грандиозный финал

Текстовая квест‑игра с выбором пути, головоломками и юмористическим сюжетом.

## Описание

Вы — ученик, случайно запертый в школе после уроков. Ваша задача — сбежать, пока вас не поймала бдительная вахтёрша. Используйте смекалку, собирайте предметы и выбирайте путь к свободе!

## Особенности

* несколько вариантов прохождения;
* система инвентаря;
* головоломки (смешивание порошков, открытие люка);
* юмористические диалоги;
* анонс следующей части в финале.
