# Побег из школы 2: Сбежать из Училища

Текстовая игра про побег ученика по имени **Тот** из строгого учебного заведения.

## Как играть

1. Установите Python 3.x.
2. Скачайте репозиторий.
3. Запустите `python game.py`.
4. Делайте выбор, вводя номер варианта (1, 2 или 3).

## Сюжет

Тот — студент Училища, где всё строго регламентировано: комендантский час, патрули, камеры. Он решает сбежать, чтобы обрести свободу.

## Особенности

* 3 главы с разными путями прохождения
* 3 концовки (хорошая, плохая, секретная)
* использование предметов (нож, верёвка, карта)
* атмосфера напряжённого побега
