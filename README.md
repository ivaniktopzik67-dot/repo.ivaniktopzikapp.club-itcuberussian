# Побег из школы 6: Сезон 2 — Экстремальный побег из дома на дереве

Текстовая приключенческая игра с таймером, головоломками и ловушками.

## Сюжет
Вы оказались заперты в загадочном доме на дереве. У вас есть всего несколько часов до заката, чтобы найти выход, решая головоломки и избегая ловушек.

## Как играть
1. Установите Python 3.x.
2. Скачайте или клонируйте репозиторий.
3. Запустите `main.py`.
4. Вводите команды текстом для взаимодействия с миром.

## Доступные команды
* `вверх` / `на чердак` — перейти на чердак;
* `вниз` / `в подвал` — спуститься в подвал;
* `осмотреться` — прочитать описание локации;
* `взять [предмет]` — подобрать предмет;
* `инвентарь` — посмотреть содержимое инвентаря;
* `выход` — завершить игру.
