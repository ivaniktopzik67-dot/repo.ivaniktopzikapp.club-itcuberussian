# Побег из тюрьмы

Текстовая приключенческая игра на Python.

## Описание

Вы заключённый, который пытается сбежать из тюрьмы. Исследуйте локации, находите предметы и используйте их, чтобы достичь свободы!

## Как играть

1. Осмотритесь и ищите предметы.
2. Передвигайтесь между локациями.
3. Используйте найденные предметы для побега.

## Команды

- `осмотреть` — осмотреть текущее место
- `искать` — поискать предметы
- `идти [место]` — перейти в другое место
- `использовать [предмет]` — использовать предмет
- `инвентарь` — посмотреть содержимое инвентаря
- `выход` — выйти из игры

## Запуск

1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте или клонируйте репозиторий.
3. Запустите игру: `python game.ru`
